# Portfolio v2

A minimal, animated personal portfolio built with Next.js 16, React 19, and Tailwind CSS 4.

## Tech Stack

- Next.js 16 + React 19
- Tailwind CSS 4
- motion/react for animations
- clsx + tailwind-merge for class utilities

## Features

- Single page scroll layout
- Dark mode support
- Fade-in animations (200ms, ease-out)
- Blue accent color theme
- Responsive design

## Sections

- **Hero** - Name, tagline, and CTA button
- **Experience** - Work history in card grid format
- **Work** - Project showcase with tech stack pills
- **Contact** - Email and social links
- **Footer** - Navigation and copyright

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view.
